== Description ==

Plugin for redirection to the logo page using cookies

== Installation ==

1. Upload `logo-page-redirect` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to /wp-admin/options-general.php?page=logo-page-redirect
